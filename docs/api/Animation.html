void animate ( el , options )
Animate an object's position, using webkit transitions.
Parameters:
el <String|Object> The CSS selector string or DOM node.
options <Object} The animation configuration Options are: properties, easing, duration, callback Example: Animating a position 100px up and 100px to the left: mui.animate('#the-element', { properties: { top: '-100px', left: '-100px', }, easing: 'ease-out', duration: '0.5s', callback: { onComplete: myFunc } > );
Returns: void
