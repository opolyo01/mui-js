Getting the Code

You can get the MUI source in a few ways, depending on your particular needs.

  • End-users who simply want the latest stable release of MUI can simply link to the library code in a script tag like so: <script src="http://yahooapis.com/mui##-min.js"></script>
  • To get a more recent development snapshot, just install the package [package name] with yinst.
  • Lastly, if you want to help develop MUI, you can check out the full source from Subversion
