Class TabView

The TabView class provides a tabbed navigation model for related contents. TabViews can be embedded inside the header of a page for global navigation, or inside the content of the page

Constructor

TabView ( el , options )
Parameters:
el <String|HTMLElement> CSS Selector or HTMLElement for the tabs container
options <Object> TabView configuration options

Properties

activeIndex - Number

The index of the currently selected tab

element - HTMLElement

The HTMLElement containing the tabs

tabContents - Array

Array of elements which are the contents for each tab

Methods

mui.TabView activateTabAtIndex ( index )

tabChanged

void tabChanged ( index )
Tab change event handler which is invoked every time a tab is activated. Override this method in implementation. The first argument to the method is the index of the selected tab.
Parameters:
index <Number> The index of the selected tab
Returns: void

tabRefreshed

void tabRefreshed ( index )
This method is invoked whenever the currently active tab is reactivated. Override this method in implementation. The first argument to the method is the index of the selected tab.
Parameters:
index <Number> The index of the selected tab
Returns: void
